To execute it in node.js:

  • Must have node.js installed
  • Open cmd and navigate to the folder containing the .js file
  • Execute it with "node" command folowed by the file name

Example:

  • C:\>cd "Path to file" (You can copy it from nav bar in file explorer)
  • C:\>Path to file\node 02-Numbers1to10.js
You can also check the result if You open Your browser's console now
